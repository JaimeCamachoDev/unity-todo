# Build

- requirement
   - os: macOS
   - unity: 2019.2.6f1

```bash
# ios ( build by xcode is needed. )
$ ./builder.sh -o /tmp/unity-todo -v 2019.2.6f1 -p ios

# android
$ ./builder.sh -o /tmp/unity-todo -v 2019.2.6f1 -p android

# mac
$ ./builder.sh -o /tmp/unity-todo -v 2019.2.6f1 -p mac
```
